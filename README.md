# 🥦 緑黄色社会 ライブ＆楽曲管理アプリ

緑黄色社会（リョクシャカ）の過去のライブ・セットリスト情報の閲覧や、自身のライブ参戦記録を管理するための専用Webアプリケーションです。
Streamlitを使用して構築され、スマホからでも快適に操作できるよう最適化されています。

## ✨ 主な機能

* **🏠 ダッシュボード**
  * 自身のライブ参戦回数や、公式楽曲の回収率（コンプリート率）を可視化。
  * 現地でよく聴いた楽曲Top5や、回収済み楽曲のレアリティ構成をグラフで表示します。
* **🎸 ライブ・フェス情報取得**
  * LiveFansから最新のライブ情報とセットリストをスクレイピングしてローカルに保存・更新します。
* **🔍 楽曲からの逆引き検索**
  * 曲名から「過去のどのライブで演奏されたか」「年毎の披露回数はどう推移しているか」を検索・グラフ化します。
* **💎 厳密な楽曲レアリティ判定**
  * 単純な披露回数ではなく、「ツアーの規模（公演数）」で重み付けをした独自の計算ロジックを採用。
  * 最終披露日や直近2年の披露率から、各楽曲を7段階（UR / SSR / SR / RR / R / UC / C）で評価します。
* **🎫 マイ参戦記録（クラウド保存対応）**
  * 自分が参戦したライブを登録し、現地で聴いた楽曲のリストを自動生成します。
  * データはGoogleスプレッドシートと連携しており、アプリ上から追加・削除したデータは永続的にクラウドに保存されます。
* **🤖 AIアシスタント**
  * Google Gemini APIを利用し、過去のデータや傾向に基づいた「次回のセトリ予想」や、自由な質問への回答を行います。

## 🛠️ 使用技術・ライブラリ

* **言語:** Python 3.x
* **フレームワーク:** Streamlit
* **データ処理・分析:** Pandas
* **スクレイピング:** Requests, BeautifulSoup4
* **グラフ描画:** Plotly (スマホでの誤操作防止のため縮尺固定設定済み)
* **クラウドデータベース連携:** gspread, st-gsheets-connection (Google Sheets API)
* **AI連携:** google-generativeai (Gemini 2.5 Flash)

## 🔒 セキュリティ・環境変数について

本アプリを動作させるために必要な機密情報（Google Cloudのサービスアカウントキー等）は、セキュリティの観点からリポジトリ内には一切含めていません。
Streamlit Community Cloud上で動作させる場合、認証情報は Streamlit の `Secrets` 機能を使用して安全に環境変数として渡しています。

※ Gemini APIキーは、アプリの画面上から都度入力して使用する仕様になっています。
